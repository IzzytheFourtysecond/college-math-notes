\hTwo Note that in the prior counter example, $K$ had a "hole" in which $f$ could hide a pole so that $f$ would have a nonzero integral. So, a question one can ask is whether that hole is the only thing preventing us from being able to generally approximate functions in $O(K)$ with polynomials?\retTwo

In this class, if $K \subseteq \mathbb{C}$ is a compact set then we define a \udefine{hole of $K$} to be a bounded connected component of $\mathbb{C} - K$.
\begin{myIndent}\pracTwo
	One note is that as $K$ is closed, $\mathbb{C} - K$ is open and thus locally connected. It follows that any hole $H$ is an open region.\retTwo

	Also note that a compact set $K \subseteq \mathbb{C}$ has no holes iff $\mathbb{C} - K$ is connected. After all, $K$ being compact implies that $\mathbb{C} - K$ has exactly one unbounded connected component. Then $K$ is hole-free iff $\mathbb{C} - K$ equals that unique unbounded component.\retTwo
\end{myIndent}

\exTwo\ul{Proposition:} If $K \subseteq \mathbb{C}$ is compact then every $f \in O(K)$ can by uniformly approximated with polynomials only if $K$ is hole-free.
\begin{myIndent}\exThreeP
	Proof:\\
	Suppose $K$ has a hole $H$ and fix $a \in H$. Then set $f(z) = \frac{1}{z - a}$ and let\\ $M = \max_{z \in K}|z  - a| > 0$. If $\{p_n\}_{n \in \mathbb{N}}$ is a sequence of polynomials uniformly\\ converging to $f$, then we can find $N$ such that $|p_N - \frac{1}{z - a}| < \frac{1}{2M}$ in $K$. In turn:
	
	{\centering $|(z - a)p_N - 1| < \frac{|z - a|}{2M} \leq \frac{1}{2}$ in $K$.\retTwo\par}

	Yet now $g(z) \coloneqq 1 - (z - a)P_N$ is a polynomial on $\overline{H}$ satisfying that $g(a) = 1$\\ and $|g(z)| < \frac{1}{2}$ on $\partial H \subseteq K$. This contradicts the maximum modulus principle. So, we've proven that if $K$ is not hole-free then there is always some function $f \in O(K)$ which cannot be uniformly approximated with polynomials. $\blacksquare$\retTwo
\end{myIndent}

\hTwo As a corollary to the theorem we are about to prove though, we can also say that the converse of the prior proposition holds. Namely:
\begin{myIndent}\exTwo
	\ul{Runge's Little's Theorem:} If $K \subseteq \mathbb{C}$ is compact and hole-free, then for all $f \in O(K)$ there exists polynomials $\{p_n\}_{n \in \mathbb{N}}$ with $p_n \UConverges f$ on $K$.\retTwo
\end{myIndent}

\pracOne\mySepTwo 
As an interesting corollary to Runge's little theorem, if $A$ is any simple proper subarc of\\ $S^1$ then we can find a polynomial $p(z)$ with $p(0) = 1$ and $|p(z)| < 1$ for all $z \in A$.
\begin{myIndent}\pracTwo
	After all we can pick a polynomial $q(z)$ with $|q(z) - \frac{1}{z}| < 1$ for all $z \in A$. In turn\\ $|1 - zq(z)| < |z| = 1$ for all $z \in A$. So, set $p(z) = 1 - zq(z)$. Then $p(0) = 1$\\ and $p(z) < 1$ for all $z \in A$.\retTwo
\end{myIndent}

\mySepTwo

\hTwo Returning to motivating Runge's theorem, perhaps our issue preventing us from getting a far reaching general theorem is that we're trying to pick from too narrow of a set of functions. If an issue we've run into is that we need our approximations to have poles in the holes, then perhaps we should instead be trying to find rational function approximations.\newpage

Namely, if $K \subseteq \mathbb{C}$ is compact and $f \in O(K)$, then we should try to uniformly approximate $f$ with a rational function whose poles are at prescribed points in the holes of $K$. In other words, we'd want to prove the following:
\begin{myIndent}\exTwo
	\ul{Theorem (almost final version):} Suppose $K \subseteq \mathbb{C}$ is compact and let $S \subseteq \mathbb{C}$ be a set of points in $\mathbb{C}$ such that $S$ contains at least one point from each hole of $K$. Then for any $f \in O(K)$ we can find a sequence of rational functions $\{R_n\}_{n \in \mathbb{N}}$ such that each $R_n$ has poles only in $S$ and $R_n \UConverges f$ on $K$.\retTwo

	\begin{myIndent}\pracTwo
		Two Remarks:
		\begin{itemize}
			\item The $R_n$ don't need to have poles at every point in $S$.
			\item If $K$ has no poles then we can take $S = \emptyset$. Then none of the $R_n$ have poles, meaning they are all polynomials. Thus, we recover Runge's little theorem.\retTwo
		\end{itemize}
	\end{myIndent}
\end{myIndent}

To get the actual theorem statement that we will prove, we replace $\mathbb{C}$ with $\mathbb{C}_\infty$.\retTwo

We say $f$ has a \udefine{pole at $\infty$} if $f(\frac{1}{z})$ has a pole at $z = 0$. 
\begin{myIndent}\pracTwo
	Note that any rational function $\frac{p(z)}{q(z)}$ has a pole at $\infty$ iff $\deg(p) > \deg(q)$. In particular,\\ [-3pt] $\frac{p(z)}{q(z)}$ only has poles at $\infty$ if $\deg(q) = 0$ and $\deg(p) > 0$. In other words, if a rational\\ function only has a pole at $\infty$, then it is a nonconstant polynomial. Similarly, we can\\ see that a rational function with no poles at all (not even at $\infty$) must be a constant\\ function.\retTwo
\end{myIndent}

\exTwo\ul{Runge's Theorem (for compact sets):} Suppose $K \subseteq \mathbb{C}$ is compact and $\widehat{S} \subseteq \mathbb{C}_\infty$ contains a single point from each connected component of $\mathbb{C}_\infty - K$. Then for all $f \in O(K)$ and $\varepsilon > 0$  there exists a a rational function $R$ with poles only in $\widehat{S}$ such that $|f - R| < \varepsilon$ on $K$.
\begin{myIndent}\exThreeP 
	\begin{myIndent}\myComment
		Hopefully it's clear that the almost final version of the theorem is recovered when we choose $\infty$ to be the element of $\widehat{S}$ in the unbounded (when restricted to $\mathbb{C}$)\\ component of $\mathbb{C}_\infty - K$.\retTwo
	\end{myIndent}

	Proof:\\
	Our general strategy will be to first establish an integral formula for compact sets. Secondly, we shall prove the existence of rational approximations without prescribed poles. Finally, we will push the poles to our desired locations.\retTwo

	\ul{Lemma 1:} Let $K \subseteq U \subseteq \mathbb{C}$ where $K$ is compact and $U$ is an open set. Then there\\ exists oriented line segments $\sigma_1, \ldots, \sigma_n$ in $U - K$ satisfying that for all holomorphic\\ functions $f \in O(U)$ we have that $f(a) = \frac{1}{2\pi i} \sum_{j=1}^n \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ for all $a \in K$.
	\begin{myIndent}\exPPP
		Proof:\\
		Since $K$ is bounded, by restricting $U$ we can without loss of generality assume that $U \neq \mathbb{C}$. This is important as it let's us then conclude that $\mathbb{C} - U$ is a nonempty closed set disjoint from $K$, and in turn $d \coloneqq \inf \{|x - y| : x \in K, y \in \mathbb{C} - U\} > 0$.\retTwo

		Tile $\mathbb{C}$ with a grid of squares of side length $\frac{d}{2\sqrt{2}}$. Then consider the closed squares $Q_1, Q_2, \ldots, Q_m$ that intersect $K$. Since $K$ is compact and thus bounded, we know that there are only finitely many such squares.\newpage
		
		\ul{Claim 1:} $K \subseteq \bigcup_{j=1}^m Q_j \subseteq U$.
		\begin{myIndent}\color{orange}
			Every point in $\mathbb{C}$ is contained in at least one square of our grid. As a result, each $k \in K$ is contained in some square $Q$. And by how we picked the $Q_j$ we then know that that $Q$ is one of those $Q_j$. So, this gives us that $K \subseteq \bigcup_{j=1}^m Q_j$.\retTwo

			Next note that $\diam(Q_j) = \sfrac{d}{2}$ for each $j$. Thus because of how we defined $d$ plus the fact that each $Q_j$ contains an element of $K$, we know that $Q_j \subseteq U$ for all $j$.\retTwo
		\end{myIndent}

		Below is a picture that hopefully makes it clear what $U$, $K$ our tiling of squares, and\\ the $Q_j$ we picked might look like.\retTwo

		{\center\includegraphics[scale=0.35]{Math_220b_Image_8.png}\retTwo\par}

		We can orient the boundary of each square in our tiling to be traveling counter-clockwise. Then, let $\sigma_1, \ldots, \sigma_n$ be the edges of $\partial Q_1, \ldots, \partial Q_m$ which are not shared by distinct squares $Q_{j_1}, Q_{j_2}$ where $1 \leq j_1 < j_2 \leq m$.\retTwo

		\ul{Claim 2:} $\bigcup_{j=1}^n \{\sigma_j\} \subseteq U - K$.
		\begin{myIndent}\color{orange}
			Note that $\{\sigma_j\} \subseteq U$ by claim 1. Meanwhile, assume $\{\sigma_j\} \cap K \neq \emptyset$ and let $k \in \{\sigma_j\} \cap K$. Since each edge in our tiling is shared by two squares, we thus have that $K$ intercepts both of those squares. And now we have a contradiction since $\{\sigma_j\}$ borders two distinct $Q_j$.\retTwo
		\end{myIndent}

		Now suppose $f \in O(U)$.\retTwo

		\ul{Claim 3:} For all $a \in U - \bigcup_{j=1}^m \partial Q_j$, we have that:

		{\centering $\sum\limits_{j=1}^m \frac{1}{2\pi i}\int_{\partial Q_j} \frac{f(z)}{z - a} \df z = \sum\limits_{j=1}^n \frac{1}{2\pi j} \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ \retTwo\par}

		\begin{myIndent}\color{orange}
			This is obvious because the common sides of the $Q_j$'s will be oriented in\\ opposite directions and thus cancel out, leaving only the integrals over the $\sigma_j$.\retTwo
		\end{myIndent}

		Note that if $a \in Q_{\ell}^\circ$, then by Cauchy's integral formula (see \inLinkRap{Math 220a Cauchy's Integral Formula (First Version)}{pages 402-404}):\newpage

		{\centering $\frac{1}{2\pi i}\int_{\partial Q_j} \frac{f(z)}{z - a}\df z = \begin{cases} f(a) & \text{ if } j = \ell \\ 0 & \text{ otherwise } \end{cases}$ \retTwo\par}

		In turn by claim 3, we can conclude for all $a \in \bigcup_{j=1}^m  Q_j^\circ$ that:

		{\centering $f(a) = \frac{1}{2\pi i} \sum\limits_{j=1}^n \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ \retTwo\par}

		Finally, note that $g(a) \coloneqq \frac{1}{2\pi i} \sum_{j=1}^n \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ is a continuous function from the set $\mathbb{C} - \bigcup_{j=1}^n \{\sigma_j\}$ (see Leibniz's rule on \inLinkRap{Leibniz Rule Analog Contour Integrals}{pages 380-381} for why $g$ is continuous). Also $g$ agrees with the continuous function $f$ on a dense subset of $(\bigcup_{j=1}^m Q_j) - (\bigcup_{j=1}^n \{\sigma_j\})$ due to the above formula. So, $f = g$ everywhere on $(\bigcup_{j=1}^m Q_j) - (\bigcup_{j=1}^n \{\sigma_j\}) \supseteq K$. This proves our lemma.\retTwo
	\end{myIndent}

	Now given any fixed $f \in O(K)$ and $\varepsilon > 0$, note by definition that $f$ can be\\ holomorphically exteded to some open set $U$ containing $K$. In turn, by the prior\\ lemma we can pick line segments $\sigma_1, \ldots, \sigma_n$ in $U - k$ satisfying for all holomorphic\\ functions $g \in O(U)$ that $g(a) = \frac{1}{2\pi i} \sum_{j=1}^n \int_{\sigma_j} \frac{g(z)}{z - a} \df z$ for all $a \in K$.\retTwo

	Define $F_j(z) \coloneqq \frac{1}{2 \pi i} \int_{\sigma_j} \frac{f(w)}{w - z}\df w$ for each $j$ and note that each $F_j$ is a holomorphic function on $\mathbb{C} - \{\sigma_j\}$ by Leibniz's rule. Our next goal will be to show that we can approximate every $F_j$ by a rational function with poles only in $\{\sigma_j\}$. Fortunately, we have the following result.\retTwo

	\ul{Lemma 2:} Suppose $K \subseteq \mathbb{C}$ is compact, $\gamma: [a, b] \to \mathbb{C}$ is a piecewise $C^1$ path (which guarentees it has finite length), and $\{\gamma\} \cap K = \emptyset$. If $f$ is a continuous function on $\{\gamma\}$ and $\varepsilon > 0$ then there is a rational function $R(z)$ with poles only in $\{\gamma\}$ such that:
	
	{\centering$\left|\int_\gamma \frac{f(w)}{w - z}\df w - R(z)\right| < \varepsilon$ for all $z \in K$\retTwo\par}

	\begin{myIndent}\exPPP
		Proof:\\
		Define $\varphi: \{\gamma\} \times K \to \mathbb{C}$ by $\varphi(w, z) = \frac{f(w)}{w - z}$. Note that $\varphi$ is well-defined because $\{\gamma\} \cap K = \emptyset$ means that we will never have that $w - z = 0$. Also note that $\varphi$ is continuous. And since $\{\gamma\} \times K$ is compact metric space (where we define the product metric $d((w_1, z_1), (w_2, z_2)) = \max(|w_1 - w_2|, |z_1 - z_2|)$), we can conclude that $\varphi$ is uniformly continuous. Hence:
		
		{\centering$\forall \varepsilon > 0,\myHS \exists \delta > 0 \suchthat \forall (w_1, z_1), (w_2, z_2) \in \{\gamma\} \times K$ we have that\\ $|w_1 - w_2| < \delta$ and $|z_1 - z_2| < \delta \Longrightarrow |\varphi(w_1, z_1) - \varphi(w_2, z_2)| < \varepsilon$.\retTwo\par}

		In particular, this means for any fixed $\varepsilon > 0$ we can find $\delta > 0$ such that for all $z \in K$ we have that $|w_1 - w_2| < \delta \Longrightarrow |\varphi(w_1, z) - \varphi(w_2, z)| < \varepsilon$.\retTwo

		But note that as $\gamma$ has finite length, we can subdivide $\gamma$ into subpaths $\gamma_1, \ldots, \gamma_\ell$\\ of length $< \delta$. 			$\left|\int_{\gamma} \frac{f(w)}{w - z}\df w - R(z)\right| = \left|\int_{\gamma}\frac{f(w)}{w - z}\df w - \sum\limits_{s = 1}^\ell \frac{f(p_s)}{p_s - z}\int_{\gamma_s} \df w\right|$\\ [14pt]

	\ul{Lemma 3:} Suppose $H \subseteq \mathbb{C}$ is connected, $K \subseteq \mathbb{C}$ is compact, and $K \cap H = \emptyset$. 
	\begin{itemize}
		\item[1.] For all $a, b \in H$ we have that $\frac{1}{z - a}$ can be uniformly approximated on $K$ by\\ polynomials in $\frac{1}{z - b}$;
		\item[2.] If $H$ is unbounded then $\frac{1}{z - a}$ can be approximated uniformly on $K$ by polynomials\\ in $z$.
	\end{itemize}

		(Proof of 1.):\\
		Fix $b$ and consider the set:
		
		{\centering\begin{tabular}{l}
			$W \coloneqq \{c \in H : \frac{1}{z - c} \text{ can be approximated uniformly on }K \text{ by polynomials in }\frac{1}{z - b}\}$.
		\end{tabular}\retTwo\par}

		Note $W \neq \emptyset$ because $b \in W$. Also, we claim for all $c \in W$ that if\\ $\delta \coloneqq \inf\{|c - k| : k \in K\}$ (and as a reminder $\delta > 0$ since $K$ is closed\\ and $c \notin K$), then $\Delta(c, \delta) \subseteq W$.
		\begin{myIndent}\color{orange}
			Why?\\
			Suppose $s \in \Delta(c, \delta)$ and consider the Laurent expansion of $\frac{1}{z - s}$ at $c$:

			{\centering\begin{tabular}{l}
				$\frac{1}{z - s} = \frac{1}{z - c} \cdot \frac{1}{1 - \frac{s - c}{z - c}} = \frac{1}{z - c} \sum\limits_{k \geq 0} \left(\frac{s - c}{z - c}\right)^k = \sum\limits_{k \geq 0} \frac{(s - c)^k}{(z - c)^{k+1}}$ if $|z - c| > |s - c|$
			\end{tabular}\retTwo\par}

			Note that as $\Delta(c, \delta) \subseteq H$ and $K \cap H = \emptyset$, we can conclude that\\ $|z - c| \geq \delta >  |s - c|$ for all $z \in K$. Furthermore, as $\frac{|s - c|}{\delta} < 1$ we know\\ that $\sum_{k \geq 0} \frac{|s - c|^k}{\delta^{k+1}} < \infty$. So by the Weierstrass $M$-test we can conclude that\\ [2pt] the above Laurent expansion converges uniformly on $K$.\retTwo

			Pick a Laurent polynomial $T$ in $\frac{1}{z - c}$ from the expansion above so that\\ $|\frac{1}{z - s} - T| < \frac{\varepsilon}{2}$ on $K$. Since $c \in W$, we know that $\frac{1}{z - c}$ can be approximated\\ by polynomials in $\frac{1}{z - b}$. The same is then true about the polynomial $T$ in $\frac{1}{z - c}$ on the compact set $K$.\newpage
			\begin{myIndent}\color{RawerSienna}
				This is because we can prove (if $K$ is a compact set) that for any\\ $f, g \in C(K)$, and sequences $\{f_n\}_{n \in \mathbb{N}}$, $\{g_n\}_{n \in \mathbb{N}}$ converging uniformly to $f$ and $g$ respectively that $f_ng_n \UConverges fg$. Also, since $f_n + g_n \UConverges f + g$ regardless of where $K$ is compact or not, we can thus prove by induction our desired claim about being able to uniformly approximate $T$.\retTwo
			\end{myIndent}

			Let $P$ be a polynomial in $\frac{1}{z - b}$ so that $|T - P| < \frac{\varepsilon}{2}$ on $K$. Then $|\frac{1}{z - s} - P| < \varepsilon$ on $K$, thus proving that $s \in W$.\retTwo
		\end{myIndent}

		It immediately follows that $W$ and $H - W$ are open subset subsets of $H$. Since $W$ is nonempty and $H$ is connected, the only way this is possible is if $W = H$.\retTwo

		(Proof of 2.):\\
		Pick $r > 0$ large enough so that $K \subseteq \Delta(0, r)$. Then by part 1 we can uniformly approximate $\frac{1}{z - a}$ on $K$ by some polynomial $T$ in $\frac{1}{z - c}$ where $|c| > r$. Then next we can consider Taylor polynomials of $T$ near $z = 0$ in order to uniformly approximate $T$ on the disk $\overline{\Delta}(0, r)$ containing $K$.\retTwo
	\end{myIndent}

	Now for all $\sigma_j$ there exists a component $H_\ell$ with $\{\sigma_j\} \subseteq H_\ell$. After all, $\{\sigma_j\}$ is a connected subset of $\mathbb{C}_\infty - K$. In turn, we can apply the prior lemma to push the poles of $R$ in $\sigma_j$ to some specified point in $\widehat{S}$. By doing this for all $j$, we complete the proof. $\blacksquare$\retTwo
\end{myIndent}

		Proof:\\
		First suppose to the contrary that $m_{\alpha ; F}(x) = g(x)h(x)$ where $g, h \in F[x] - \{0\}$ and $\deg(g), \deg(h) < \deg(m_{\alpha; F})$. Yet note that:

		{\centering $0 = m_{\alpha ; F}(\alpha) = g(\alpha)h(\alpha) \Longrightarrow g(\alpha) = 0 \text{ or } h(\alpha) = 0$.\retTwo\par}

		Without loss of generality assume $g(\alpha) = 0$. Then $g(x) = m_{\alpha ; F}(x)k(x)$ for some $k(x) \in F[x]$. Since $m_{\alpha; F}(x)$ is not the zero polynomial and $m_{\alpha; F}(x) = g(x)h(x)$, we know that $g(x)$ is not the zero polynomial. In turn, $k(x)$ is also not the zero\\ polynomial. But now we have a contradiction as $\deg(g) \geq \deg(m_{\alpha; F})$. This proves claim 1.\retTwo

		As for claim 2, since $p(\alpha) = 0$ we know that $p(x) = m_{\alpha;F}(x)h(x)$ for some\\ $h(x) \in F[x]$. But then as $p$ is irreducible, we know that $h$ must be a constant. Finally, as both $p$ and $m_{\alpha; F}$ are monic, the constant $h$ must be $1$. So, $p = m_{\alpha;F}$. $\blacksquare$\retTwo
	\end{myIndent}
\end{enumerate}

\ul{Theorem:} If $\alpha \in E$ is algebraic over $F$, then:\\ [-20pt]
\begin{enumerate}
	\item[1.] $F[\alpha]$ is a field.\\ [-20pt]
	\item[2.] $(1, \alpha, \ldots, \alpha^{d-1})$ is an $F$-basis of $F[\alpha]$ over $F$ where $d = \deg(m_{\alpha; F})$.
	\begin{myIndent}\exThreeP
		Proof:\\
		Consider the evaluation homomorphism $e_\alpha: F[x] \to E$. Then by the first\\ isomorphism theorem, we have that:

		{\centering $F[\alpha] = \myIm(e_\alpha) \cong \frac{F[x]}{\ker(e_\alpha)} = \frac{F[x]}{\langle m_{\alpha ; F}\rangle}$ \retTwo\par}

		Yet as $m_{\alpha ; F}$ is irreducible and all ideals of $F[x]$ are principal, we can conclude that $\langle m_{\alpha; F}\rangle$ is a maximal ideal. So, $\frac{F[x]}{\langle m_{\alpha ; F}\rangle}$ is a field.\retTwo

		Next, note by the long division theorem that every element of $\frac{F[x]}{\langle m_{\alpha ; F}\rangle}$ has a unique representive of the form $\sum_{i=0}^{d-1} a_i x^i$ where $d = \deg(m_{\alpha; F})$. Thus, $(1, \overline{x}, \ldots, \overline{x^{d-1}})$ is\\ [3pt] an $F$-basis of this quotient.
		\begin{myIndent}\myComment
			As a side note, this paragraph also works for any commutative unital ring $A$ and monic polynomial $f \in A[x]$ for getting an $A$-basis for $\frac{A[x]}{\langle f\rangle}$.\retTwo
		\end{myIndent}

		In turn, by applying the evaluation homomorphism we get that $(1, \alpha, \ldots, \alpha^{d-1})$ is an $F$-basis of $F[\alpha]$. $\blacksquare$\newpage
	\end{myIndent}
\end{enumerate}

\hTwo We say a field extension $E/F$ is a \udefine{finite extension} if $\dim_F(E) < \infty$, and we denote\\ $\dim_F(E) = [E : F]$.\retTwo

Meanwhile, $E/F$ is said to be an \udefine{algebraic extension} if all $\alpha \in E$ are algebraic over $F$.

\exTwo\ul{The Tower Lemma:} $[K : F] = [K : E][E : F]$.
\begin{myIndent}\exThreeP
	Proof:\\
	Suppose $\{e_i\}_{i \in I}$ is an $F$-basis of $E$ and $\{k_j\}_{j \in J}$ is an $E$-basis of $K$. Then we claim that $\{e_i k_j\}_{(i, j) \in I \times J}$ is an $F$-basis of $K$.
	\begin{myIndent}\exPPP
		After all, for all $k \in K$ there exists $c_1, \ldots, c_n \in E$ and $j_1, \ldots, j_n \in J$ such that $K = c_1 k_{j_1} + \ldots + c_n k_{j_n}$. Next, for all $s$ we can find $f_1^{(s)}, \ldots, f_{m_s}^{(s)} \in F$ and $i_1^{(s)}, \ldots, i_{m_s}^{(s)} \in I$ with $c_s = f_1^{(s)}e_{i_1^{(s)}} + \ldots + f_{m_s}^{(s)} e_{i_{m_s}^{(s)}}$. And now:

		{\centering $k = \sum_{s=1}^n \sum_{\ell = 1}^{m_s}f_\ell^{(s)} \cdot e_{i_\ell^{(s)}}k_{j_s}$ \retTwo\par}

		This proves our proposed basis spans all of $K$.\retTwo

		Meanwhile to show linear independence, suppose:
		
		{\centering$\sum_{i, j} f_{i,j} e_j k_j = \sum_{j} (\sum_i f_{i,j} e_i) k_j = 0$.\retTwo\par}

		As the $k_j$ are $E$-linearly independent, we can thus conclude that $\sum_i f_{i,j} e_i = 0$ for\\ every fixed $j$. Next, because the $e_i$ are $F$-linearly independent, we can conclude\\ $f_{i,j} = 0$ for all $i$ and $j$. \exThreeP $\blacksquare$\retTwo
	\end{myIndent}
\end{myIndent}

\exTwo\ul{Theorem:} If $f \in F[x] - F$ is irreducible, then there exists a field extension $E/F$ and element $\alpha \in E$ such that $E = F[\alpha]$ and $f(\alpha) = 0$.
\begin{myIndent}\exThreeP
	Proof:\\
	Let $E \coloneqq \frac{F[x]}{\langle f\rangle}$. Since $F[x]$ is a PID and $f$ is irreducible, $\langle f\rangle$ is a maximal ideal. Hence, $E$ is a field. Also note that $i: F \hookrightarrow E$ given by $i(c) = c + \langle f\rangle$ is an injective unital ring homomorphism. Thus, $E/F$ is a field extension.\retTwo

	Let $\alpha \coloneqq x + \langle f\rangle \in E$. Then as $\frac{F[x]}{\langle f\rangle}$ is generated by $i(F)$ and $\alpha$, we have that $E = F[\alpha]$. Also note that $f(\alpha) = f + \langle f\rangle = 0 + \langle f\rangle$. $\blacksquare$\retTwo
\end{myIndent}

\exTwo\ul{Theorem (Existence of Splitting Fields):} Suppose $f(x) \in F[x] - F$. Then there\\ exists $(E/F, [\alpha_1, \ldots, \alpha_d])$ with $d = \deg(f)$. In other words:
\begin{itemize}
	\item[1.] $E/F$ is a field extensions, $\alpha_1, \ldots, \alpha_d \in E$, and $E = F[\alpha_1, \ldots, \alpha_d]$.
	\item[2.] $f(x) = \lt(f) \prod_{i=1}^d (x-\alpha_i)$. 
\end{itemize}

$E$ is called a \udefine{splitting field} of $f$ over $F$.
\begin{myIndent}\exThreeP
	Proof:\\
	We proceed by induction on $d = \deg(f)$. For our base case, if $d = 1$ then we can just set $E = F$ and $\alpha$ to be the zero of $f$ (which is guarenteed to exist).\retTwo
	
	Meanwhile, suppose $d > 1$ and let $f_1(x) \in F[x]$ be an irreducible factor of $f(x)$. Then by a theorem on the prior page there exists a field extension $(E/F, \alpha_1)$ such that $E_1 = F[\alpha_1]$ and $f_1(\alpha_1) = 0$. Yet then as $f(\alpha_1) = 0$, we can conclude there exists $g(x) \in E_1[x]$ such that $f(x) = (x - \alpha_1)g(x)$. Also note that $\lt(f) = \lt(g)$ and $\deg(g) = d - 1$.\retTwo

	By the induction hypothesis, there exists a field extension $(E/E_1, [\alpha_2, \ldots, \alpha_d])$ such that\\ $E = E_1[\alpha_2, \ldots, \alpha_d]$ and $g(x) = \lt(f)\prod_{i=2}^d (x - \alpha_i)$. It follows that $E = F[\alpha_1, \ldots, \alpha_d]$ and $f(x) = (x-\alpha_1)g(x) = \lt(f)\prod_{i=1}^d (x-\alpha_i)$. $\blacksquare$\newpage
\end{myIndent}

\ul{Theorem (Uniqueness of Splitting Fields):} Suppose $\theta: F \xrightarrow{\sim} F^\prime$ and $f \in F[x]$. If\\ $(E/F, [\alpha_1, \ldots, \alpha_d])$ is a splitting field of $f$ and $(E^\prime / F^\prime, [\alpha_1^\prime, \ldots, \alpha_d^\prime])$ is a splitting\\ field of $f^\theta$, then there exists $\widehat{\theta} \in \Isom_\theta(E, E^\prime)$.
\begin{myIndent}\exThreeP
	\begin{myIndent}\myComment
		In particular, this shows splitting fields are unique up to isomorphism.\retTwo
	\end{myIndent}

	Proof:\\
	We again proceed by induction on $d = \deg(f)$. Like before, if $d = 1$ then the problem is trivial. Any degree $1$ polynomial in $F[x]$ has a zero in $F$. Similarly, any degree $1$ polynomial in $F^\prime[x]$ has a zero in $F^\prime$. Thus, we'd have that $F[\alpha_1] = F$ and $F^\prime[\alpha_1^\prime] = F^\prime$. In turn, $\theta$ gives us a $\theta$-isomorphism between $E$ and $E^\prime$.\retTwo
	
	Now suppose $d > 1$ and let $f_1$ be an irreducible factor of $f$. Then note that $f_1^\theta$ is an\\ irreducible factor of $f^\theta$. After all, isomorphisms must send factors to factors and irreducible elements to irreducible elements. Since $f_1 \divides f$ and $f(x) = \lt(f)\prod_{i=1}^d (x - \alpha_i)$, there must exists $i_0$ such that $f_1(\alpha_{i_0}) = 0$. Without loss of generality we may assume $f_1(\alpha_1) = 0$.\\ Similarly, as $f_1^{\theta} \divides f^\theta$ and $f^{\theta}(x) = \theta(\lt(f))\prod_{j=1}^d (x- \alpha_j^\prime)$, there must exist $j_0$ with\\ $f_1^{\theta}(\alpha^\prime_{j_0}) = 0$, and without loss of generality we may say that $j_0 = 1$.\retTwo

	Let $E_1 \coloneqq F[\alpha_1]$ and $E_1^\prime \coloneqq F^\prime[\alpha_1^\prime]$. By a prior theorem, there exists a unique\\ $\theta_1 \in \Isom_\theta(E_1, E_1^\prime)$ with $\theta_1(\alpha_1) = \alpha^\prime$. Then since $f_1(\alpha_1) = 0$, we can conclude\\ $f(\alpha_1) = 0$ and thus there exists $h \in E_1[x]$ with $f(x) = (x - \alpha_1)h(x)$. Also, we can\\ then see that $f^\theta(x) = f^{\theta_1}(x) = (x - \alpha_1^\prime)h^{\theta_1}(x)$ and that $\deg(h)= d - 1$\retTwo

	Now $(E/E_1, [\alpha_2, \ldots, \alpha_d])$ is a splitting field of $h$ and $(E^\prime/E_1^\prime, [\alpha_2^\prime, \ldots, \alpha_d^\prime])$ is a splitting field of $h^{\theta_1}$. Therefore, by our induction hypothesis we can conclude that there is a\\ $\theta_1$-isomorphism $\widehat{\theta}: E \xrightarrow{\sim} E^\prime$. Yet we then also have that $\widehat{\theta} \in \Isom_\theta(E, E^\prime)$.\retTwo
\end{myIndent}

Moreover, any such $\widehat{\theta}$ sends $[\alpha_1, \ldots, \alpha_d]$ to $[\alpha_1^\prime, \ldots, \alpha_d^\prime]$.
\begin{myIndent}\exThreeP
	Proof:\\
	If $f \in F[x]$ and $\alpha \in E$ satisfies that $f(\alpha) = 0$, then we must have that $f^\theta(\widehat{\theta}(\alpha)) = 0$ as well. Similarly, we have that if $f^\theta(\alpha^\prime) = 0$, then $f(\widehat{\theta}^{-1}(\alpha^\prime)) = 0$. $\blacksquare$\retTwo
\end{myIndent}

\hTwo We call $(E/F, [\alpha_1, \ldots, \alpha_d])$ the \udefine{splitting data} of a polynomial $f$.

\blab{Set 8 Problem 7:} Prove that $\mathbb{Q}[\sqrt{2}]$ and $\mathbb{Q}[\sqrt{3}]$ are not isomorphic (as fields).
\begin{myIndent}\HexOne
	Suppose for the sake of contradiction that $\theta: \mathbb{Q}[\sqrt{2}] \to \mathbb{Q}[\sqrt{3}]$ is a field isomorphism, then note that since $\theta(1) = 1$ and $\theta$ preserves sums and differences, we can conclude that $\theta(n) = n$ for all $n \in \mathbb{Z}$. 	Next, consider that $\theta(\sqrt{2})^2 = \theta(2) = 2$. 	{\centering $p^2 + 2pq\sqrt{3} + 3q^2 = 2$ \retTwo\par}

	\begin{itemize}
		\item If $q = 0$ then we must have that $p^2 = 2$. Yet this is impossible as no $p \in \mathbb{Q}$ satisfies that $p^2 = 2$.
		\item If $p = 0$ then we have that $q^2 = \frac{2}{3}$ Yet once again no $q \in \mathbb{Q}$ satisfies this.
	\end{itemize}

	Thus we conclude no such $\theta$ can exist. $\blacksquare$\retTwo

Question, can we go in reverse? Given any power of a prime number $q = p^d$, is there a finite field of size $q$?
\begin{myIndent}\hThree

	\begin{myIndent}\exFourP
		Suppose $\alpha, \beta \in V_{x^q - x}(E)$. Then $\alpha^q = \alpha$ and $\beta^q = \beta$ implies that $(\alpha \beta)^q = \alpha\beta$. Also if\\ $\alpha \neq 0$ then $(\alpha^{-1})^q = \alpha^{-1}$. Hence $\alpha\beta \in V_{x^q - x}(E)$ and $\alpha^{-1} \in V_{x^q - x}(E) - \{0\}$.\retTwo

		{\centering\begin{tabular}{l}
			$(\alpha + \beta)^p = \sum\limits_{k=0}^p \binom{p}{k}\alpha^k \beta^{p - k} = \alpha^p + \beta^p + \sum\limits_{k=1}^{p-1} \binom{p}{k}\alpha^k \beta^{p-k}$\\ [12pt]
			$\phantom{(\alpha + \beta)^p = \sum\limits_{k=0}^p \binom{p}{k}\alpha^k \beta^{p - k}} = \alpha^p + \beta^p + \sum\limits_{k=1}^{p-1} \frac{p}{k} \cdot \binom{p-1}{k-1}\alpha^k \beta^{p-k} = \alpha^p + \beta^p + 0$
		\end{tabular}\retTwo\par}

		\begin{myIndent}\color{orange}
			In other words, the map $x \mapsto x^p$ is a unital ring homomorphism in any field (or integral domain) with characteristic $p$.\retTwo
		\end{myIndent}

		{\centering$(\alpha + \beta)^q = \alpha^q + \beta^q = \alpha + \beta$.\newpage\par}

	Finally, we ask how many elements $E$ has.\retTwo
\end{myIndent}

A polynomial $f \in F[x]$ is called \udefine{separable} if it has distinct zeros in its splitting field.\retTwo

Also given any field $F$ and $f \in F[x]$, we define the \udefine{formal derivative} of $f = \sum_{i=0}^n c_i x^i$\\ by $\frac{\df }{\df x} f(x) = f^\prime(x) \coloneqq \sum_{i=0}^{n-1} (i+1) c_{i+1} x^i$.
\begin{myIndent}\pracTwo
	Note that $\frac{\df}{\df x} : F[x] \to F[x]$ is an $F$-linear map. Also you can prove (although I won't here) that $\frac{\df}{\df x}$ has the typical product, power, and chain rules from normal calculus.\retTwo
\end{myIndent}

Using formal derivatives, we can come up with a nifty test for if a polynomial $f \in F[x]$\\ is separable. First, notice the following:
\begin{itemize}
	\item Suppose $f(x) = (x - \alpha)^2 h(x)$. Then $f^\prime(x) = 2(x - \alpha)h(x) + (x - \alpha)^2 h^\prime(x)$,\\ thus implying that $f^\prime(\alpha) = 0$.
	\item Meanwhile suppose $f(x) = (x - \alpha)g(x)$ where $g(\alpha) \neq 0$. Then\\ $f^\prime(x) = g(x) + (x - \alpha)g^\prime(x)$, thus implying that $f^\prime(\alpha) = g(\alpha) \neq 0$.\retTwo
\end{itemize}

Therefore for any $\alpha \in V_f(E)$ where $E$ is a splitting field of $f$, we have that $(x - \alpha)^2 \divides f$\\ iff $f^\prime(\alpha) = 0$. Yet also note that $f^\prime(\alpha) = 0$ and $f(\alpha) = 0$ iff $m_{\alpha; F} \in F[x]$ is a common divisor of $f$ and $f^\prime$. This proves that if $f$ is not separable, then $\gcd(f, f^\prime) \neq 1$.\retTwo

Conversely, suppose $h = \gcd(f, f^\prime) \neq 1$. Then as $h \divides f$, we know that $h(\alpha) = 0$ for some $\alpha \in V_f(E)$. In turn, $f^\prime(\alpha) = 0$ as well. So by our prior reasoning, we  know $(x - \alpha)^2$ divides $f$, thus showing that $f$ is not separable.\retTwo

Thus, we have the following theorem:
\begin{myIndent}
	$f \in F[x]$ is separable if and only if $\gcd(f, f^\prime) = 1$.\retTwo

	\begin{myIndent}\myComment
		Note that this theorem is useful because $\gcd(f, f^\prime)$ can be calculated without\\ knowing anything about the splitting field of $f$.
	\end{myIndent}
\end{myIndent}

Also conversely, we've shown that we can get a finite field of size $p^d$ by considering the splitting field of $x^{p^d} - x$ over $\mathbb{F}_p$. This finishes establishing the existence of finite fields.\retTwo

As for the uniqueness of finite fields, we have the following theorem:\retTwo

\exTwo\ul{Theorem:} If $F_1$ and $F_2$ are both finite fields with $q = p^d$ many elements, then there is\\ a field isomorphism $\theta: F_1 \xrightarrow{\sim} F_2$.

\end{myIndent}

	\item[(c)] $E/F$ is a Galois extension.\newpage
\end{enumerate}

	\item $K_1 \subseteq K_2 \Longrightarrow \Aut_{K_1}(E) \supseteq \Aut_{K_2}(E) \Longrightarrow \Psi(K_1) \geq \Psi(K_2)$.\retTwo
\end{itemize}

Also by a prior theorem plus the first observation above, we know that:

{\centering$[E : \Fix(G)] \geq |\Aut_{\Fix(G)}(E)| \geq |G|$ for all $G \in \Sub(\Aut_F(E))$.\retTwo\par}

\exTwo\ul{Theorem:} We also have that $|G| \geq [E : \Fix(G)]$.
\begin{myIndent}\exThreeP
	Proof:\\
	For all $e_1, \ldots, e_{n+1}$ where $n = |G|$, we must show that there exists\\ $(c_1, \ldots, c_{n+1}) \in \Fix(G)^{n+1} - \{(0, \ldots, 0)\}$ with $c_1 e_1 + \ldots + c_{n+1} e_{n+1} = 0$.\retTwo

	\ul{Lemma:} Suppose $\{0\} \neq V \subseteq E^m$ is an $E$-subspace. Also suppose $G \curvearrowright E^m$\\ componentwise via field-automorphisms and $V$ is $G$-invariant. Then the set $V^G$ of\\ $G$-fixed points in $V$ is nonempty.
	\begin{myIndent}\exPPP
		Proof:\\
		Let $v_0 \in V - \{0\}$ be a vector with smallest support (i.e. fewest nonzero coordinates). Without loss of generality, we may assume the first component of $v_0$ is nonzero. Then after multiplying by the inverse of the first component, we may assume that\\ $v_0 = (1, c_1, \ldots, c_m)$.\retTwo

		Now $v_0 \in V^G$. After all, for all $g \in G$ we have that $(g \cdot v_0) - v_0 \in V$ and has a smaller support then $v_0$. This is because $g$ will send the $0$ coordinates to $0$ and the $1$ coordinates to $1$. It follows by the minimality of our choice of $v_0$ that $(g \cdot v_0) - v_0 = 0$. In turn, $g \cdot v_0 = v_0$ for all $g \in G$.\retTwo
	\end{myIndent}

	Now write $G = \{1_G, g_2, \ldots, g_n\}$ and set:

	{\centering $V \coloneqq \left\{ (c_1, \ldots, c_{n+1}) \in E^{n+1}\hspace{1em} : \hspace{1em} \begin{matrix} c_1 e_1 + \ldots + c_{n+1}e_{n+1} = 0 \\ c_1 g_2(e_1) + \ldots + c_{n+1}g_2(e_{n+1}) = 0 \\ \vdots \\ c_1 g_n(e_1) + \ldots + c_{n+1}g_n(e_{n+1}) = 0\end{matrix}\right\}$\retTwo\par}

	Note that $V$ is $G$-invariant. After all, if $(c_1, \ldots, c_{n+1}) \in V$ then:

	{\centering\begin{tabular}{l}
		$g(c_1)g_i(e_1) + \ldots + g(c_{n+1})g_i(e_{n+1}) = g(c_1(g^{-1}g_i)(e_1) + \ldots + c_{n+1}(g^{-1}g_i)(e_{n+1}))$\\ [4pt]
		$\phantom{g(c_1)g_i(e_1) + \ldots + g(c_{n+1})g_i(e_{n+1})} = g(0) = 0$ for all $g_i \in G$.
	\end{tabular} \retTwo\par}

	Yet also note that $V$ is the kernel of an $n$x$(n+1)$ matrix. It follows that $\dim_E(V) \geq 1$. So $V \neq \{0\}$ and in turn $V^G \neq \emptyset$ by our lemma above.\newpage

	Finally, note that $V^G \subseteq \Fix(G)^{n+1}$. Thus, there exists $(c_1, \ldots, c_{n+1}) \in \Fix(G) \cap V$. And by the first linear equation in the definition of $V$, this proves that $e_1, \ldots, e_{n+1}$ are linearly dependent over $\Fix(G)$. $\blacksquare$\retTwo
\end{myIndent}

\exTwo\ul{Corollary:} If $G \in \Sub(\Aut_F(E))$, then:
\begin{enumerate}
	\item[(a)] $E/\Fix(G)$ is a Galois extension.
	\item[(b)] $G = \Aut_{\Fix(G)}(E)$.
	
	\begin{myIndent}\exThreeP
		Proof:\\
		Since $|G| = [E: \Fix(G)] \geq |\Aut_{\Fix(G)}(E)| \geq |G|$, we know that:
		\begin{itemize}
			\item $|\Aut_{\Fix(G)}(E)| = [E : \Fix(G)]$. Hence $E/\Fix(G)$ is a Galois extension.\\ [-6pt]
			
			\item $|G| = |\Aut_{\Fix(G)}(E)|$. And since we also know that $G \leq \Aut_{\Fix(G)}(E)$, this proves that $G = \Aut_{\Fix(G)}(E)$. $\blacksquare$\retTwo
		\end{itemize}
	\end{myIndent}
\end{enumerate}

\exTwo\ul{Theorem:} Given any $K \in \Int(E/F)$, if $E/K$ is a Galois extension then:

{\centering$(\Phi \circ \Psi)(K) = \Fix(\Aut_K(E)) = K$.\retTwo\par}

\begin{myIndent}\exThreeP
	Proof:\\
	I already noted prior that $K \subseteq \Fix(\Aut_K(E))$. Thus, all we need to show is that\\ $K \supseteq \Fix(\Aut_K(E))$. Fortunately, because $E/K$ is Galois and $K \subseteq \Fix(\Aut_K(E))$\\ we know that:
	
	{\centering$|\Aut_K(E)| = [E : K] = [E : \Fix(\Aut_K(E))] \cdot [\Fix(\Aut_K(E)) : K]$\retTwo\par}

	{\centering $[\Fix(\Aut_K(E)) : K] = 1$ \retTwo\par}

	Hence, $K = \Fix(\Aut_K(E))$. $\blacksquare$\retTwo
\end{myIndent}
