\hTwo Note that in the prior counter example, $K$ had a "hole" in which $f$ could hide a pole so that $f$ would have a nonzero integral. So, a question one can ask is whether that hole is the only thing preventing us from being able to generally approximate functions in $O(K)$ with polynomials?\retTwo

In this class, if $K \subseteq \mathbb{C}$ is a compact set then we define a \udefine{hole of $K$} to be a bounded connected component of $\mathbb{C} - K$.
\begin{myIndent}\pracTwo
	One note is that as $K$ is closed, $\mathbb{C} - K$ is open and thus locally connected. It follows that any hole $H$ is an open region.\retTwo

	Also note that a compact set $K \subseteq \mathbb{C}$ has no holes iff $\mathbb{C} - K$ is connected. After all, $K$ being compact implies that $\mathbb{C} - K$ has exactly one unbounded connected component. Then $K$ is hole-free iff $\mathbb{C} - K$ equals that unique unbounded component.\retTwo
\end{myIndent}

\exTwo\ul{Proposition:} If $K \subseteq \mathbb{C}$ is compact then every $f \in O(K)$ can by uniformly approximated with polynomials only if $K$ is hole-free.
\begin{myIndent}\exThreeP
	Proof:\\
	Suppose $K$ has a hole $H$ and fix $a \in H$. Then set $f(z) = \frac{1}{z - a}$ and let\\ $M = \max_{z \in K}|z  - a| > 0$. If $\{p_n\}_{n \in \mathbb{N}}$ is a sequence of polynomials uniformly\\ converging to $f$, then we can find $N$ such that $|p_N - \frac{1}{z - a}| < \frac{1}{2M}$ in $K$. In turn:
	
	{\centering $|(z - a)p_N - 1| < \frac{|z - a|}{2M} \leq \frac{1}{2}$ in $K$.\retTwo\par}

	Yet now $g(z) \coloneqq 1 - (z - a)P_N$ is a polynomial on $\overline{H}$ satisfying that $g(a) = 1$\\ and $|g(z)| < \frac{1}{2}$ on $\partial H \subseteq K$. This contradicts the maximum modulus principle. So, we've proven that if $K$ is not hole-free then there is always some function $f \in O(K)$ which cannot be uniformly approximated with polynomials. $\blacksquare$\retTwo
\end{myIndent}

\hTwo As a corollary to the theorem we are about to prove though, we can also say that the converse of the prior proposition holds. Namely:
\begin{myIndent}\exTwo
	\ul{Runge's Little's Theorem:} If $K \subseteq \mathbb{C}$ is compact and hole-free, then for all $f \in O(K)$ there exists polynomials $\{p_n\}_{n \in \mathbb{N}}$ with $p_n \UConverges f$ on $K$.\retTwo
\end{myIndent}

\pracOne\mySepTwo 
As an interesting corollary to Runge's little theorem, if $A$ is any simple proper subarc of\\ $S^1$ then we can find a polynomial $p(z)$ with $p(0) = 1$ and $|p(z)| < 1$ for all $z \in A$.
\begin{myIndent}\pracTwo
	After all we can pick a polynomial $q(z)$ with $|q(z) - \frac{1}{z}| < 1$ for all $z \in A$. In turn\\ $|1 - zq(z)| < |z| = 1$ for all $z \in A$. So, set $p(z) = 1 - zq(z)$. Then $p(0) = 1$\\ and $p(z) < 1$ for all $z \in A$.\retTwo
\end{myIndent}

\mySepTwo

\hTwo Returning to motivating Runge's theorem, perhaps our issue preventing us from getting a far reaching general theorem is that we're trying to pick from too narrow of a set of functions. If an issue we've run into is that we need our approximations to have poles in the holes, then perhaps we should instead be trying to find rational function approximations.\newpage

Namely, if $K \subseteq \mathbb{C}$ is compact and $f \in O(K)$, then we should try to uniformly approximate $f$ with a rational function whose poles are at prescribed points in the holes of $K$. In other words, we'd want to prove the following:
\begin{myIndent}\exTwo
	\ul{Theorem (almost final version):} Suppose $K \subseteq \mathbb{C}$ is compact and let $S \subseteq \mathbb{C}$ be a set of points in $\mathbb{C}$ such that $S$ contains at least one point from each hole of $K$. Then for any $f \in O(K)$ we can find a sequence of rational functions $\{R_n\}_{n \in \mathbb{N}}$ such that each $R_n$ has poles only in $S$ and $R_n \UConverges f$ on $K$.\retTwo

	\begin{myIndent}\pracTwo
		Two Remarks:
		\begin{itemize}
			\item The $R_n$ don't need to have poles at every point in $S$.
			\item If $K$ has no poles then we can take $S = \emptyset$. Then none of the $R_n$ have poles, meaning they are all polynomials. Thus, we recover Runge's little theorem.\retTwo
		\end{itemize}
	\end{myIndent}
\end{myIndent}

To get the actual theorem statement that we will prove, we replace $\mathbb{C}$ with $\mathbb{C}_\infty$.\retTwo

We say $f$ has a \udefine{pole at $\infty$} if $f(\frac{1}{z})$ has a pole at $z = 0$. 
\begin{myIndent}\pracTwo
	Note that any rational function $\frac{p(z)}{q(z)}$ has a pole at $\infty$ iff $\deg(p) > \deg(q)$. In particular,\\ [-3pt] $\frac{p(z)}{q(z)}$ only has poles at $\infty$ if $\deg(q) = 0$ and $\deg(p) > 0$. In other words, if a rational\\ function only has a pole at $\infty$, then it is a nonconstant polynomial. Similarly, we can\\ see that a rational function with no poles at all (not even at $\infty$) must be a constant\\ function.\retTwo
\end{myIndent}

\exTwo\ul{Runge's Theorem (for compact sets):} Suppose $K \subseteq \mathbb{C}$ is compact and $\widehat{S} \subseteq \mathbb{C}_\infty$ contains a single point from each connected component of $\mathbb{C}_\infty - K$. Then for all $f \in O(K)$ and $\varepsilon > 0$  there exists a a rational function $R$ with poles only in $\widehat{S}$ such that $|f - R| < \varepsilon$ on $K$.
\begin{myIndent}\exThreeP 
	\begin{myIndent}\myComment
		Hopefully it's clear that the almost final version of the theorem is recovered when we choose $\infty$ to be the element of $\widehat{S}$ in the unbounded (when restricted to $\mathbb{C}$)\\ component of $\mathbb{C}_\infty - K$.\retTwo
	\end{myIndent}

	Proof:\\
	Our general strategy will be to first establish an integral formula for compact sets. Secondly, we shall prove the existence of rational approximations without prescribed poles. Finally, we will push the poles to our desired locations.\retTwo

	\ul{Lemma 1:} Let $K \subseteq U \subseteq \mathbb{C}$ where $K$ is compact and $U$ is an open set. Then there\\ exists oriented line segments $\sigma_1, \ldots, \sigma_n$ in $U - K$ satisfying that for all holomorphic\\ functions $f \in O(U)$ we have that $f(a) = \frac{1}{2\pi i} \sum_{j=1}^n \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ for all $a \in K$.
	\begin{myIndent}\exPPP
		Proof:\\
		Since $K$ is bounded, by restricting $U$ we can without loss of generality assume that $U \neq \mathbb{C}$. This is important as it let's us then conclude that $\mathbb{C} - U$ is a nonempty closed set disjoint from $K$, and in turn $d \coloneqq \inf \{|x - y| : x \in K, y \in \mathbb{C} - U\} > 0$.\retTwo

		Tile $\mathbb{C}$ with a grid of squares of side length $\frac{d}{2\sqrt{2}}$. Then consider the closed squares $Q_1, Q_2, \ldots, Q_m$ that intersect $K$. Since $K$ is compact and thus bounded, we know that there are only finitely many such squares.\newpage
		
		\ul{Claim 1:} $K \subseteq \bigcup_{j=1}^m Q_j \subseteq U$.
		\begin{myIndent}\color{orange}
			Every point in $\mathbb{C}$ is contained in at least one square of our grid. As a result, each $k \in K$ is contained in some square $Q$. And by how we picked the $Q_j$ we then know that that $Q$ is one of those $Q_j$. So, this gives us that $K \subseteq \bigcup_{j=1}^m Q_j$.\retTwo

			Next note that $\diam(Q_j) = \sfrac{d}{2}$ for each $j$. Thus because of how we defined $d$ plus the fact that each $Q_j$ contains an element of $K$, we know that $Q_j \subseteq U$ for all $j$.\retTwo
		\end{myIndent}

		Below is a picture that hopefully makes it clear what $U$, $K$ our tiling of squares, and\\ the $Q_j$ we picked might look like.\retTwo

		{\center\includegraphics[scale=0.35]{Math_220b_Image_8.png}\retTwo\par}

		We can orient the boundary of each square in our tiling to be traveling counter-clockwise. Then, let $\sigma_1, \ldots, \sigma_n$ be the edges of $\partial Q_1, \ldots, \partial Q_m$ which are not shared by distinct squares $Q_{j_1}, Q_{j_2}$ where $1 \leq j_1 < j_2 \leq m$.\retTwo

		\ul{Claim 2:} $\bigcup_{j=1}^n \{\sigma_j\} \subseteq U - K$.
		\begin{myIndent}\color{orange}
			Note that $\{\sigma_j\} \subseteq U$ by claim 1. Meanwhile, assume $\{\sigma_j\} \cap K \neq \emptyset$ and let $k \in \{\sigma_j\} \cap K$. Since each edge in our tiling is shared by two squares, we thus have that $K$ intercepts both of those squares. And now we have a contradiction since $\{\sigma_j\}$ borders two distinct $Q_j$.\retTwo
		\end{myIndent}

		Now suppose $f \in O(U)$.\retTwo

		\ul{Claim 3:} For all $a \in U - \bigcup_{j=1}^m \partial Q_j$, we have that:

		{\centering $\sum\limits_{j=1}^m \frac{1}{2\pi i}\int_{\partial Q_j} \frac{f(z)}{z - a} \df z = \sum\limits_{j=1}^n \frac{1}{2\pi j} \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ \retTwo\par}

		\begin{myIndent}\color{orange}
			This is obvious because the common sides of the $Q_j$'s will be oriented in\\ opposite directions and thus cancel out, leaving only the integrals over the $\sigma_j$.\retTwo
		\end{myIndent}

		Note that if $a \in Q_{\ell}^\circ$, then by Cauchy's integral formula (see \inLinkRap{Math 220a Cauchy's Integral Formula (First Version)}{pages 402-404}):\newpage

		{\centering $\frac{1}{2\pi i}\int_{\partial Q_j} \frac{f(z)}{z - a}\df z = \begin{cases} f(a) & \text{ if } j = \ell \\ 0 & \text{ otherwise } \end{cases}$ \retTwo\par}

		In turn by claim 3, we can conclude for all $a \in \bigcup_{j=1}^m  Q_j^\circ$ that:

		{\centering $f(a) = \frac{1}{2\pi i} \sum\limits_{j=1}^n \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ \retTwo\par}

		Finally, note that $g(a) \coloneqq \frac{1}{2\pi i} \sum_{j=1}^n \int_{\sigma_j} \frac{f(z)}{z - a}\df z$ is a continuous function from the set $\mathbb{C} - \bigcup_{j=1}^n \{\sigma_j\}$ (see Leibniz's rule on \inLinkRap{Leibniz Rule Analog Contour Integrals}{pages 380-381} for why $g$ is continuous). Also $g$ agrees with the continuous function $f$ on a dense subset of $(\bigcup_{j=1}^m Q_j) - (\bigcup_{j=1}^n \{\sigma_j\})$ due to the above formula. So, $f = g$ everywhere on $(\bigcup_{j=1}^m Q_j) - (\bigcup_{j=1}^n \{\sigma_j\}) \supseteq K$. This proves our lemma.\retTwo
	\end{myIndent}

	Now given any fixed $f \in O(K)$ and $\varepsilon > 0$, note by definition that $f$ can be\\ holomorphically exteded to some open set $U$ containing $K$. In turn, by the prior\\ lemma we can pick line segments $\sigma_1, \ldots, \sigma_n$ in $U - k$ satisfying for all holomorphic\\ functions $g \in O(U)$ that $g(a) = \frac{1}{2\pi i} \sum_{j=1}^n \int_{\sigma_j} \frac{g(z)}{z - a} \df z$ for all $a \in K$.\retTwo

	Define $F_j(z) \coloneqq \frac{1}{2 \pi i} \int_{\sigma_j} \frac{f(w)}{w - z}\df w$ for each $j$ and note that each $F_j$ is a holomorphic function on $\mathbb{C} - \{\sigma_j\}$ by Leibniz's rule. Our next goal will be to show that we can approximate every $F_j$ by a rational function with poles only in $\{\sigma_j\}$. Fortunately, we have the following result.\retTwo

	\ul{Lemma 2:} Suppose $K \subseteq \mathbb{C}$ is compact, $\gamma: [a, b] \to \mathbb{C}$ is a piecewise $C^1$ path (which guarentees it has finite length), and $\{\gamma\} \cap K = \emptyset$. If $f$ is a continuous function on $\{\gamma\}$ and $\varepsilon > 0$ then there is a rational function $R(z)$ with poles only in $\{\gamma\}$ such that:
	
	{\centering$\left|\int_\gamma \frac{f(w)}{w - z}\df w - R(z)\right| < \varepsilon$ for all $z \in K$\retTwo\par}

	\begin{myIndent}\exPPP
		Proof:\\
		Define $\varphi: \{\gamma\} \times K \to \mathbb{C}$ by $\varphi(w, z) = \frac{f(w)}{w - z}$. Note that $\varphi$ is well-defined because $\{\gamma\} \cap K = \emptyset$ means that we will never have that $w - z = 0$. Also note that $\varphi$ is continuous. And since $\{\gamma\} \times K$ is compact metric space (where we define the product metric $d((w_1, z_1), (w_2, z_2)) = \max(|w_1 - w_2|, |z_1 - z_2|)$), we can conclude that $\varphi$ is uniformly continuous. Hence:
		
		{\centering$\forall \varepsilon > 0,\myHS \exists \delta > 0 \suchthat \forall (w_1, z_1), (w_2, z_2) \in \{\gamma\} \times K$ we have that\\ $|w_1 - w_2| < \delta$ and $|z_1 - z_2| < \delta \Longrightarrow |\varphi(w_1, z_1) - \varphi(w_2, z_2)| < \varepsilon$.\retTwo\par}

		In particular, this means for any fixed $\varepsilon > 0$ we can find $\delta > 0$ such that for all $z \in K$ we have that $|w_1 - w_2| < \delta \Longrightarrow |\varphi(w_1, z) - \varphi(w_2, z)| < \varepsilon$.\retTwo

		But note that as $\gamma$ has finite length, we can subdivide $\gamma$ into subpaths $\gamma_1, \ldots, \gamma_\ell$\\ of length $< \delta$. For each $1 \leq s \leq \ell$ pick $p_s \in \{\gamma_s\}$ and set $c_s = f(p_s) \int_{\gamma_s} \df w$.\\ [-2pt] Finally, define $R(z) = \sum_{s=1}^\ell \frac{c_s}{p_s - z}$.\retTwo
		
		Clearly, $R$ is a rational function with roots only in $\{\gamma\}$. (In particular $R$ doesn't have a root at $\infty$.)\newpage

		Also note for all $z \in K$ that:

		{\centering\begin{tabular}{l}
			$\left|\int_{\gamma} \frac{f(w)}{w - z}\df w - R(z)\right| = \left|\int_{\gamma}\frac{f(w)}{w - z}\df w - \sum\limits_{s = 1}^\ell \frac{f(p_s)}{p_s - z}\int_{\gamma_s} \df w\right|$\\ [14pt]
			$\phantom{\left|\int_{\gamma} \frac{f(w)}{w - z}\df w - R(z)\right|} = \left| \sum\limits_{s = 1}^\ell \int_{\gamma_s}\left(\frac{f(w)}{w - z} - \frac{f(p_s)}{p_s - z}\right)\df w \right|$\\ [14pt]
			$\phantom{\left|\int_{\gamma} \frac{f(w)}{w - z}\df w - R(z)\right|} \leq \sum\limits_{s = 1}^\ell \int_{\gamma_s}|\varphi(w, z) - \varphi(p_s, z)|\df |w| < \sum\limits_{s = 1}^\ell \varepsilon \cdot \mathrm{length}(\gamma_s)$\\ [4pt]
			$\phantom{\left|\int_{\gamma} \frac{f(w)}{w - z}\df w - R(z)\right| \leq \sum\limits_{s = 1}^\ell \int_{\gamma_s}|\varphi(w, z) - \varphi(p_s, z)|\df |w|} = \varepsilon \cdot \mathrm{length}(\gamma)$.
		\end{tabular}\retTwo\par}
	\end{myIndent}

	By applying the prior lemma, for all $1 \leq j \leq n$ we can find a rational function $R_j$ with poles only in $\{\sigma_j\}$ and satisfying that $|F_j - R_j| < \sfrac{\varepsilon}{2n}$ on $K$. Then because $f = \sum_{j=1}^n F_j$ we can conclude that $R = \sum_{j=1}^n R_j$ is a rational function with poles only in $\bigcup_{j=1}^n \{\sigma_j\}$ and satisfying that $|F - R| < \sfrac{\varepsilon}{2}$ on $K$.\retTwo

	Finally, we shall "push" the poles of $R$ from $\bigcup_{j=1}^n \{\sigma_j\}$ to $\widehat{S}$.\retTwo

	Write $\mathbb{C}_\infty - K = \bigcup_{\ell} H_\ell$ where each $H_\ell$ is a connected component of $\mathbb{C}_\infty - K$.\retTwo

	\ul{Lemma 3:} Suppose $H \subseteq \mathbb{C}$ is connected, $K \subseteq \mathbb{C}$ is compact, and $K \cap H = \emptyset$. 
	\begin{itemize}
		\item[1.] For all $a, b \in H$ we have that $\frac{1}{z - a}$ can be uniformly approximated on $K$ by\\ polynomials in $\frac{1}{z - b}$;
		\item[2.] If $H$ is unbounded then $\frac{1}{z - a}$ can be approximated uniformly on $K$ by polynomials\\ in $z$.
	\end{itemize}

	\begin{myIndent}\exPPP
		(Proof of 1.):\\
		Fix $b$ and consider the set:
		
		{\centering\begin{tabular}{l}
			$W \coloneqq \{c \in H : \frac{1}{z - c} \text{ can be approximated uniformly on }K \text{ by polynomials in }\frac{1}{z - b}\}$.
		\end{tabular}\retTwo\par}

		Note $W \neq \emptyset$ because $b \in W$. Also, we claim for all $c \in W$ that if\\ $\delta \coloneqq \inf\{|c - k| : k \in K\}$ (and as a reminder $\delta > 0$ since $K$ is closed\\ and $c \notin K$), then $\Delta(c, \delta) \subseteq W$.
		\begin{myIndent}\color{orange}
			Why?\\
			Suppose $s \in \Delta(c, \delta)$ and consider the Laurent expansion of $\frac{1}{z - s}$ at $c$:

			{\centering\begin{tabular}{l}
				$\frac{1}{z - s} = \frac{1}{z - c} \cdot \frac{1}{1 - \frac{s - c}{z - c}} = \frac{1}{z - c} \sum\limits_{k \geq 0} \left(\frac{s - c}{z - c}\right)^k = \sum\limits_{k \geq 0} \frac{(s - c)^k}{(z - c)^{k+1}}$ if $|z - c| > |s - c|$
			\end{tabular}\retTwo\par}

			Note that as $\Delta(c, \delta) \subseteq H$ and $K \cap H = \emptyset$, we can conclude that\\ $|z - c| \geq \delta >  |s - c|$ for all $z \in K$. Furthermore, as $\frac{|s - c|}{\delta} < 1$ we know\\ that $\sum_{k \geq 0} \frac{|s - c|^k}{\delta^{k+1}} < \infty$. So by the Weierstrass $M$-test we can conclude that\\ [2pt] the above Laurent expansion converges uniformly on $K$.\retTwo

			Pick a Laurent polynomial $T$ in $\frac{1}{z - c}$ from the expansion above so that\\ $|\frac{1}{z - s} - T| < \frac{\varepsilon}{2}$ on $K$. Since $c \in W$, we know that $\frac{1}{z - c}$ can be approximated\\ by polynomials in $\frac{1}{z - b}$. The same is then true about the polynomial $T$ in $\frac{1}{z - c}$ on the compact set $K$.\newpage
			\begin{myIndent}\color{RawerSienna}
				This is because we can prove (if $K$ is a compact set) that for any\\ $f, g \in C(K)$, and sequences $\{f_n\}_{n \in \mathbb{N}}$, $\{g_n\}_{n \in \mathbb{N}}$ converging uniformly to $f$ and $g$ respectively that $f_ng_n \UConverges fg$. Also, since $f_n + g_n \UConverges f + g$ regardless of where $K$ is compact or not, we can thus prove by induction our desired claim about being able to uniformly approximate $T$.\retTwo
			\end{myIndent}

			Let $P$ be a polynomial in $\frac{1}{z - b}$ so that $|T - P| < \frac{\varepsilon}{2}$ on $K$. Then $|\frac{1}{z - s} - P| < \varepsilon$ on $K$, thus proving that $s \in W$.\retTwo
		\end{myIndent}

		It immediately follows that $W$ and $H - W$ are open subset subsets of $H$. Since $W$ is nonempty and $H$ is connected, the only way this is possible is if $W = H$.\retTwo

		(Proof of 2.):\\
		Pick $r > 0$ large enough so that $K \subseteq \Delta(0, r)$. Then by part 1 we can uniformly approximate $\frac{1}{z - a}$ on $K$ by some polynomial $T$ in $\frac{1}{z - c}$ where $|c| > r$. Then next we can consider Taylor polynomials of $T$ near $z = 0$ in order to uniformly approximate $T$ on the disk $\overline{\Delta}(0, r)$ containing $K$.\retTwo
	\end{myIndent}

	Now for all $\sigma_j$ there exists a component $H_\ell$ with $\{\sigma_j\} \subseteq H_\ell$. After all, $\{\sigma_j\}$ is a connected subset of $\mathbb{C}_\infty - K$. In turn, we can apply the prior lemma to push the poles of $R$ in $\sigma_j$ to some specified point in $\widehat{S}$. By doing this for all $j$, we complete the proof. $\blacksquare$\retTwo
\end{myIndent}

Using entirely topological arguments, we can also formulate a version of Runge's theorem for open sets.\retTwo

\exTwo\ul{Theorem:} Suppose $U \subseteq \mathbb{C}$ is an open set and $\widehat{S} \subseteq \mathbb{C}_\infty - U$ is a set containing at least\\ [2pt] one point from each connected component of $\mathbb{C}_\infty - U$. If $f \in O(U)$ then there exists a\\ [-3pt] sequence of rational functions $\{R_n\}_{n \in \mathbb{N}}$ with poles only in $\widehat{S}$ such that $R_n \locUConverges f$ on $U$.
\begin{myIndent}\exThreeP
	Proof:\\
	We start by proving the following lemma. The only nontrivial part of the lemma (which makes me unable to merely cite a result from math 240b) is proving part 4.\retTwo

	\ul{Lemma:} For any $U \subseteq \mathbb{C}$ we can find a sequence of compact sets $K_n \subseteq U$ with:
	\begin{itemize}
		\item[1.] $U = \bigcup_{n \geq 1} K_n$,
		\item[2.] $K_n \subseteq K_{n+1}^\circ$,
		\item[3.] for all compact $K \subseteq U$ there exists $n $ with $K \subseteq K_n$,
		\item[4.] \textit{each} connected component of $\mathbb{C}_\infty - K_n$ contains a component of $\mathbb{C}_\infty - U$.
	\end{itemize}
	\begin{myIndent}\exPPP
		Proof:\\
		If $U = \mathbb{C}$ then this lemma is easy. Just set $K_n = \overline{\Delta}(0, n)$ for all $n$. Hence, from now on we can assume without loss of generality that $U \neq \mathbb{C}$.\retTwo

		Let $K_n = \{z \in \mathbb{C} : |z| \leq n$ and $\inf\{| z - w| : w \in \mathbb{C} - U\} \geq \frac{1}{n}\}$ for all $n$.
		\begin{itemize}
			\item Note that if $z \in K_n$ then because $\inf\{|z - w| : w \in \mathbb{C} - U\} \geq \frac{1}{n} > 0$,\\ we know that $z \notin \mathbb{C}  - U$. This proves that $K_n \subseteq U$ for each $n$.\newpage
			
			\item Also note that if $z \in U$, then because $\inf\{|z - w| : w \in \mathbb{C} - U\} > 0$ since $\mathbb{C} - U$ is a disjoint closed set, we can find $n$ with $\frac{1}{n}$ less than that infimum. Also by increasing $n$ we can additionally get that $|z| \leq \frac{1}{n}$. Thus $z \in K_n$ and we've proven that $U = \bigcup_{n \geq 1} K_n$.\retTwo
			
			\item Each $K_n$ is clearly bounded. Also note that each $K_n$ is closed since:
			
			{\center $\mathbb{C} - K_n = \{z : |z| > n\} \cup \bigcup_{b \in \mathbb{C} - U}\Delta(b, \frac{1}{n})$\\ [4pt]\par}
			
			is an arbitrary union of open sets. This proves each $K_n$ is compact.\retTwo

			\item Suppose $z \in K_n$ and let $r < \frac{1}{n} - \frac{1}{n+1}$. Then for all $w \in \Delta(z, r)$, we have that:
			\begin{itemize}
				\item[$\circ$] $|w| \leq |z| + |w - z| \leq n + r < n + 1$.
				\item[$\circ$] For all $a \in \mathbb{C} - U$ we have that $|z - a| \leq |w - z| + |w - a|$ implies that:
				
				{\centering $|w - a| \geq |z - a| - |w - z| \geq \frac{1}{n} - r$ \retTwo\par}

				It follows that $\inf\{|w - a| : a \in \mathbb{C} - U\} \geq \frac{1}{n} - r > \frac{1}{n+1}$.\retTwo
			\end{itemize}

			As a result, $\Delta(z, r) \subseteq K_{n+1}$, thus proving that $z \in K_{n+1}^\circ$. THis proves that $K_n \subseteq K_{n+1}^\circ$ for all $n$.\retTwo
			
			\item Note that if $K \subseteq U$ is any compact set, then the $K_{n}^\circ$ form an open cover of $K$. \lparen After all, by our prior bullet point we know that:
			
			{\centering $U = \bigcup_{n \geq 1} K_n \subseteq \bigcup_{n \geq 2} K_n^\circ \subseteq \bigcup_{n \geq 2} K_n \subseteq U$\rparen.\retTwo\par}
			
			It follows by the compactness of $K$ that only finitely many of them are needed to cover $K$. Yet also as $\{K_n^\circ\}$ is an increasing sequence of sets, this in turn means that there is some $n$ with $K \subseteq K_n^\circ \subseteq K_n$.\retTwo

			\item Let $A = \mathbb{C}_\infty - K_n$ and $B = \mathbb{C}_\infty - U$. Then note that $A \supseteq B$. We claim that every connected component of $A$ contains a connected component of $B$.
			\begin{myIndent}\color{orange}
				Why?\\
				Let $A_c$ be a component of $A$. Then firstly note that $\infty \in A \cap B$. As\\ a result, if $A_c$ is the component of $A$ containing $\infty$, we can let $B_c$\\ be the component of $B$ also containing $\infty$. Then as $A_c \cap B_c \neq \emptyset$\\ and $B_c \subseteq B \subseteq A$, we can conclude that $B_c \subseteq A_c$.\retTwo

				Meanwhile, suppose $\infty \notin A_c$ and recall from when we were showing $K_n$ to be closed that:

				{\center $A = (\{z : |z| > n\} \cup \{\infty\}) \cup \bigcup\limits_{\substack{b \in B \\ b \neq \infty}}\Delta(b, \frac{1}{n})$\retTwo\par}

				Now $A_c$ can't be disjoint from every set $\Delta(b, \frac{1}{n})$ for all $b \in B - \{\infty\}$.\\ After all, otherwise we'd have to have that $A_c$ is contained in the connected set $(\{z : |z| > n\} \cup \{\infty\})$ which contains $\infty$. Yet this contradicts that $\infty \notin A_c$.\retTwo

				As a result, there exists $b \in B - \{\infty\}$ with $A_c \cap \Delta(b, \frac{1}{n}) \neq \emptyset$. And since $\Delta(b, \frac{1}{n}) \subseteq A$ and is a connected set, this implies that $\Delta(b, \frac{1}{n}) \subseteq A_c$.\newpage

				To finish off, let $B_c$ be the component of $B$ containing $b$. Then since\\ $b \in B_c \cap A_c$ and $B_c \subseteq B \subseteq A$, we can see that $B_c \subseteq A_c$.\retTwo
			\end{myIndent}
		\end{itemize}
	\end{myIndent}

	Using the prior topological lemma plus Runge's theorem for compact sets, hopefully it's now clear how to prove our desired theorem.\retTwo

	Suppose $f \in O(U)$ and let $\{K_n\}_{n \in \mathbb{N}}$ be an increasing sequence of compact sets in $U$ constructed via the prior lemma. By applying Runge's theorem for compact sets to $f$ and $K_n$, we can get a rational function $R_n$ with poles only in $\widehat{S}$ and which satisfies that\\ $|f - R_n| < \frac{1}{n}$ on $K_n$. It is easy to then check that $R_n \locUConverges f$ on $U$. $\blacksquare$\retTwo
\end{myIndent}
